# KL Plomb Service Futur SARL

Site vitrine Next.js App Router pour KL Plomb Service Futur SARL.

## Développement

```bash
npm install
npm run dev
```

## Déploiement Vercel

- Framework preset : Next.js
- Build command : `npm run build`
- Output directory : valeur par défaut Next.js
- Root directory : racine du dépôt
